# ggzp
一个仿照boss直聘的react前后端项目

# You should support mongodb database locally if you need to run this project.

# Download
git clone https://github.com/Bruiser-pink/ggzp.git

# Open mongodb server

# init web project
npm install -save

# init server project
npm install -save

# Run server project
npm start

# Run web project
npm start
